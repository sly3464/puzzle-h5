# 移动端拼图游戏

一个基于 HTML5 的移动端拼图小游戏，支持触摸操作、自定义图片和多种难度级别。

## 功能特点

- 🎮 支持移动端触摸操作，体验流畅
- 🎯 多种难度级别（3x3, 4x4, 5x5）
- 📸 支持自定义图片上传
- ⏱️ 实时计时和步数统计
- 🏆 排行榜系统记录最佳成绩
- 💾 支持游戏进度保存
- 🎵 音效反馈增强游戏体验

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- jQuery UI Touch Punch (触摸支持)
- LocalStorage (数据持久化)

## 安装和使用

1. 克隆仓库：https://github.com/sly3464/puzzle-h5.git
2. 打开 index.html 文件即可开始游戏
